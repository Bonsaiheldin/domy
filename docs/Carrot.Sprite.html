Sprites are game objects which contain the actual HTML elements for rendering.
Parameters:
Name Type Attributes Default Description
game Carrot.Game The core game object.
x number <optional>
 0 The x coordinate in the world of the sprite.
y number <optional>
 0 The y coordinate in the world of the sprite.
key string <optional>
 null This is the image for the sprite. If left empty, the sprite will be just a green rectangle.
frame number <optional>
 0 The starting frame of the image (only for spritesheets). If left empty, it will be null.
group Carrot.Group <optional>
 null The group this sprite shall be added to. If left empty, it will be added directly to the world container
