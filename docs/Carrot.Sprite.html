Class: Sprite

.Carrot. Sprite

Carrot.Sprite

new Sprite(game [, x] [, y] [, key] [, frame] [, group])

Sprites are game objects which contain the actual HTML elements for rendering.
Parameters:
Name Type Argument Default Description
game Carrot.Game The core game object.
x number <optional>
 0 The x coordinate in the world of the sprite.
y number <optional>
 0 The y coordinate in the world of the sprite.
key string <optional>
 null This is the image for the sprite. If left empty, the sprite will be just a green rectangle.
frame number <optional>
 0 The starting frame of the image (only for spritesheets). If left empty, it will be null.
group Carrot.Group <optional>
 null The group this sprite shall be added to. If left empty, it will be added directly to the world container

Methods

destroy()

Destroys the sprite and removes it from its group and the game world.

kill()

Kills the sprite. Not much more than a placeholder for now. Later this will be used as a soft destroy() to enable object pools.

setAnimation(type)

Sets a CSS animation for this sprite
Parameters:
Name Type Description
type string The name of the animation

setFrame( [frame])

Changes the frame shown. Only for spritesheets.
Parameters:
Name Type Argument Default Description
frame number <optional>
 0

setGlow( [blur] [, spread] [, color] [, inset])

Applies a glow effect on the sprite. Its shape is determined by the sprite's body which can be a rectangle or a circle.
Parameters:
Name Type Argument Default Description
blur number <optional>
 0 Blur in pixels.
spread number <optional>
 0 Spread in pixels.
color Carrot.Color | string <optional>
 "#00ff00" The color of the glow. Must be given in one of the following formats: Hexadecimal, RGB, RGBA, HSL, HSLA or one of the 140 predefined browser colors.
inset boolean <optional>
 null Defines if the glow should be go out or inside the sprite.
