Class: Sprite

Sprite(game, x, y, key, frame, group)

new Sprite(game, x, y, key, frame, group)

Sprites are game objects which contain the actual HTML elements for rendering.
Parameters:
Name Type Description
game Domy.Game Your global game object.
x number The x coordinate in the world of the sprite.
y number The y coordinate in the world of the sprite.
key string This is the image for the sprite. If left empty, the sprite will be just a green rectangle.
frame string The starting frame of the image (only for spritesheets). If left empty, it will be null.
group Domy.Group The group this sprite shall be added to. If left empty, it will be added directly to the world container
